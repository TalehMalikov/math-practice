# math-practice

Python notebooks for practicing applied math — series, calculus, and linear algebra, with reusable utilities and visualizations.

## Structure

```
math-practice/
├── series/
│   ├── taylor.ipynb
│   ├── fourier.ipynb
│   └── power_series.ipynb
├── calculus/
│   ├── derivatives.ipynb
│   ├── integrals.ipynb
│   └── odes.ipynb
├── linear_algebra/
│   ├── matrices.ipynb
│   └── eigenvalues.ipynb
└── utils/
    ├── plotting.py
    └── symbolic.py
```

## Setup

```bash
pip install -r requirements.txt
```

## Topics
- **Series** — Taylor, Fourier, Power Series
- **Calculus** — Derivatives, Integrals, ODEs
- **Linear Algebra** — Matrices, Eigenvalues
